Display recent comments list simply on sidebar sort by entry.

== Description ==

Entry based recent comments.

== Installation ==

Simply place the widget_recent_comments_by_entry.php file in your /wp-content/plugins/ directory
and activate through the administration panel, and then go to the widget panel and
drag it to where you would like to have it!

== Frequently Asked Questions ==

= What's difference between default one? =

Default one shows recent comments by comment-driven which means those are not categorized, but this one categorized by each entry. The entry which have newest comments will be displayed on top.
